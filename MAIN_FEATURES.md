# 📋 Λειτουργίες της Εφαρμογής

1️⃣ **Αρχική Εγκατάσταση (Setup)**
* [ΑΦΑΙΡΕΘΗΚΕ] Δεν γίνεται πλέον στον υπολογιστή — όλη η διαχείριση μέσω web/cloud

2️⃣ **Σύστημα Χρηστών**
* Admin: Πλήρης διαχείριση συστήματος
* User: Περιορισμένη πρόσβαση
* Δημιουργία/Διαγραφή χρηστών
* Authentication με JWT tokens

3️⃣ **Διαχείριση License**
* Έλεγχος ισχύος άδειας (1, 6, ή 12 μήνες)
* Προειδοποιήσεις λήξης (30 ημέρες πριν)
* Ανανέωση άδειας
* Status: Active/Warning/Expired

4️⃣ **Ανέβασμα Δεδομένων Tracking**
Το σύστημα δέχεται 2 τύπους CSV αρχείων:

**A) iButton CSV (Οδηγοί)**
Date, Driver, iButton, Address, Latitude, Longitude
2024-01-15, John Doe, 123456789, Athens Greece, 37.9838, 23.7275

**B) Vehicle CSV (Οχήματα)**
Date, Vehicle, Address, Latitude, Longitude
2024-01-15, AB-1234-XY, Thessaloniki, 40.6401, 22.9444

5️⃣ **Αναφορές (Reports)**
* Αναφορά Ταξιδιών (Trips Report)
  - Εμφανίζει όλα τα ταξίδια ανά:
    * Οδηγό
    * Όχημα
    * Ημερομηνία
  - Δεδομένα: Διεύθυνση, GPS συντεταγμένες, χρόνος

* Αναφορά Ωρών Εργασίας (Work Hours Report)
  - Υπολογίζει ώρες εργασίας ανά οδηγό
  - Βασίζεται στα iButton records
  - Εξαγωγή σε PDF/Excel

6️⃣ **Χειροκίνητη Προσθήκη Ταξιδιού**
* Προσθήκη ταξιδιού χωρίς CSV
* Πεδία:
  - Τύπος (iButton/Vehicle)
  - Ημερομηνία
  - Οδηγός/Όχημα
  - Διεύθυνση
  - GPS συντεταγμένες

7️⃣ **Backup & Restore**
* Backup: Εξαγωγή όλων των δεδομένων (users, records, license)
* Restore: Επαναφορά από backup αρχείο
* Format: ZIP με JSON files

8️⃣ **Admin Dashboard - Διαχείριση Εταιρειών & Συνδρομών** (Νοέμβριος 2025)
* **Διαχείριση Εταιρειών**
  - Πίνακας με βασικά στοιχεία εταιρειών (όνομα, συνδρομή, πλήθος οχημάτων/χρηστών, status)
  - Αναζήτηση και φίλτρα ανά κατάσταση
  - Μαζικές ενέργειες (batch update, batch renew, batch disable)
  - Export σε Excel/PDF
  
* **Διαχείριση Οχημάτων**
  - Πίνακας οχημάτων ανά εταιρεία
  - Στοιχεία: μοντέλο, πινακίδα, status, ημερομηνίες συνδρομής/λήξης
  - Φίλτρα ανά εταιρεία και κατάσταση
  - Μαζικές ενέργειες και export
  
* **Διαχείριση Συνδρομών**
  - Πίνακας συνδρομών με τύπο, τιμή, όριο οχημάτων
  - Ημερομηνίες λήξης και alerts για λήξεις
  - Αυτόματη ανανέωση (auto-renew)
  - Στατιστικά: ενεργές, λήγουν σύντομα, έχουν λήξει
  
* **Σύστημα Ειδοποιήσεων**
  - Χειροκίνητη αποστολή ενημερώσεων σε εταιρείες/χρήστες
  - Αυτόματες ειδοποιήσεις για λήξη άδειας (15/30 ημέρες πριν)
  - Αυτόματες ειδοποιήσεις για υπέρβαση ορίου οχημάτων
  - Log/ιστορικό ειδοποιήσεων (ποιος, πότε, τι στάλθηκε)
  - Email templates για κάθε τύπο ειδοποίησης
  
* **Σημειώσεις Υλοποίησης**
  - Dummy data για επίδειξη λειτουργικότητας
  - Mockup email service (console.log για demo)
  - Διαθέσιμο μόνο σε admin users
  - Πλήρης ενσωμάτωση με υπάρχον Admin Dashboard
